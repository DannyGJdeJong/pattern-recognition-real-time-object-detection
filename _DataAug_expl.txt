The classes 'Pedestrian', 'trafficLight-Red', 'trafficLight-Green', 'truck', 'biker' were important and underrepresented classes in the data set.
The size of these classes is increased via data augmention. The combined file is a merge of the standard annotations and the newly generated images.
- increase red/green color value
- gausian blur

The total data set increased from 15k to 30k images
The class representation is now:
- car			64k
- Pedestrian		32k
- trafficLight-Red	20k
- trafficLight-Green	16k
- truck			2k
- biker			5k
